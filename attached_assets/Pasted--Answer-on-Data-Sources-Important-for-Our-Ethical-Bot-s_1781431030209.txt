✅ Answer on “Глаз Бога” Data Sources (Important for Our Ethical Bot)
Глаз Бога started as an OSINT aggregator (open-source intelligence) but quickly evolved into something much broader. According to multiple investigations and statements from its creator:
Main Sources (What They Claimed Publicly)
	•	Official public registries — ЕГРЮЛ / ЕГРИП (companies/INN/OGRN), ФНС, ГИБДД (vehicles), Росреестр, etc.
	•	Public social media profiles — VK, OK, Instagram, Telegram usernames, Avito, etc. (scraping open pages).
	•	Other open data — Search engines, public leaks that were already dumped online, phone books from apps like GetContact.
Reality (What Actually Powered It)
The bulk of its impressive “probiv” capabilities came from purchased or exchanged leaked databases (leaks from banks, mobile operators, delivery services like Yandex Eda, insurance companies, etc.). These were bought on dark web forums or through middlemen. It aggregated thousands of sources, including old operator bases, marketing databases, and user-uploaded contacts.0
This is why it felt so powerful — it mixed legal public data + illegal/private leaks.

We cannot and will not touch any leaked/private data. We stay 100% on free public sources only. This keeps the bot safe, promotable in normal Russian communities, and focused on value + natural promo for fuel-tickets-ru.

async def ip_lookup(update: Update, context: ContextTypes.DEFAULT_TYPE):
    if not context.args:
        await update.message.reply_text("Использование: /ip 8.8.8.8")
        return
    ip = context.args[0]
    await update.message.reply_text(f"🔍 Инфо по IP: {ip}...")
    try:
        # Free public IP info (no key for basic)
        resp = requests.get(f"https://ipapi.co/{ip}/json/", timeout=10)
        if resp.status_code == 200:
            data = resp.json()
            result = (
                f"🌍 Страна: {data.get('country_name', 'N/A')}\n"
                f"🏙 Город: {data.get('city', 'N/A')}\n"
                f"Провайдер: {data.get('org', 'N/A')}"
            )
            await update.message.reply_text(result)
        else:
            raise Exception
    except:
        await update.message.reply_text("Инфо по IP не найдена (публичные источники).")
    await send_promo(update)
